Native Avatar System

ชุด SVG Avatar สำหรับแอพจัดการแจ้งเตือน Domain — 12 ตัวละครน่ารัก พร้อมระบบเปลี่ยนสีพื้นหลังได้

🎨 เลือกสีพื้นหลัง (Background Color)

📋 วิธีใช้งาน SVG Avatars

โครงสร้าง SVG

ทุก SVG มี class .avatar-bg สำหรับวงกลมพื้นหลัง คุณสามารถเปลี่ยนสีผ่าน CSS หรือ JavaScript ได้:

/* เปลี่ยนสีพื้นหลังทั้งหมด */ .avatar-bg { fill: #FF5722; } /* หรือ JavaScript */ document.querySelectorAll('.avatar-bg').forEach(el => { el.style.fill = '#4CAF50'; });

รายการ Avatar ทั้งหมด (12 ตัว)

  • bear.svg - หมีน้ำตาล (Brown Bear)
  • cat.svg - แมวส้ม (Orange Cat)
  • rabbit.svg - กระต่าย (Bunny)
  • dog.svg - สุนัขคอร์กี้ (Corgi Dog)
  • elephant.svg - ช้างฟ้า (Blue Elephant)
  • owl.svg - นกฮูก (Owl)
  • penguin.svg - เพนกวิน (Penguin)
  • panda.svg - แพนด้า (Panda)
  • fox.svg - จิ้งจอก (Fox)
  • monkey.svg - ลิง (Monkey)
  • dragon.svg - มังกรน้อย (Dragon)
  • bird.svg - นกน้อย (Bird)

ใช้งานใน React/Vue/Angular

// React Component Example import { ReactComponent as BearAvatar } from './avatars/bear.svg'; const UserAvatar = ({ bgColor }) => ( <div style={{ '--bg': bgColor }}> <BearAvatar /> </div> );

คุณสมบัติ

  • SVG Vector 100% - รองรับทุกขนาดไม่แตก
  • เปลี่ยนสีพื้นหลังง่ายผ่าน CSS class .avatar-bg
  • ขนาดไฟล์เล็ก (แต่ละไฟล์ ~2-4 KB)
  • รองรับ Dark Mode สามารถสลับสีพื้นหลังได้ตามธีม
  • ใช้ Inline SVG เพื่อให้ควบคุมสีผ่าน CSS/JS ได้
